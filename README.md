1. Clone the repository:
```bash
git clone https://github.com/amal5haji/ai-companions.git
```

2. Install dependencies:
```bash
cd ai-companions
npm install
```

3. Set up environment variables:
```bash
cp .env.example .env.local
```

4. Run the development server:
